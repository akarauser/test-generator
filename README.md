# test-generator
Data Replication Script

This script replicates a training folder structure and randomly selects a specified percentage of images to move into test folders. It utilizes a configuration file for settings and logging for debugging. The script handles directory creation, file moving, and random image selection, ensuring data integrity and efficient test dataset generation. It employs random shuffling for unbiased image selection and includes comprehensive logging for monitoring its execution and identifying potential issues. The core functionality involves moving image files based on a configured ratio to a designated test folder.
